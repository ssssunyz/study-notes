## Memory (LangGraph)
### InMemory Memory
普通llm是<mark>stateless</mark>的

<img src="./pics/截屏2026-05-19 下午3.05.25.png" alt="Alt Text" width="700"> <br>

https://docs.langchain.com/oss/python/langchain/short-term-memory
- short-term memory: 当前chat的context
- long-term memory: cross chat experiences & knowledge

LangChain的短期记忆是通过`AgentState`来实现的。chat history也是`AgentState`的一部分。

LangGraph提供了`Checkpointer`来保存`AgentState`，每次用户与AI的交互都会生成一个snapshot，记录为一个checkpoint，可以用来support <mark>time travel</mark>。同一个chat里的多个checkpoint用同一个`thread_id`标记。

To add short-term memory (thread-level persistence) to an agent, you need to specify a checkpointer when creating an agent.
