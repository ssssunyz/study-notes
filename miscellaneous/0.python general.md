```python
from A import B
```
> from a module/package, import specific names (classes, functions, variables, constant, module)

A:
- module(模块) — 一个 .py 文件,比如 from math import sqrt 中的 math
- package(包) — 包含多个 module 的目录,比如 `from langchain.chains import RetrievalQA` 中的 `langchain`(整体)和 `langchain.chains`(子模块路径)

B:
- 通常叫 name 或 symbol — 因为它可以是任何东西: class、function、variable、constant,甚至是另一个 module
- 比如 from fastapi import FastAPI(class)、from os import path(submodule)、from math import pi(constant)
